■ゲームタイトル
ランダムクエスト

■ゲーム概要
ランダムなパーティでいきなりボスと戦うターン制バトルゲームです。
味方パーティーの人数は4人で、それぞれのキャラは職業や技などがランダムに決まります。
ボスを倒せばゲームクリアとなります。

■ランダム要素
ゲーム開始時に味方1人1人のキャラメイクが自動的にランダムで行われます。
以下の要素がランダムです。
(1)職業
(2)キャラレベル
(3)特技（職業で固定のものも有り）
(4)魔法（職業で固定のものも有り）
(5)アイテム
(6)装備
(7)キャラ属性（装備によって決まる）
